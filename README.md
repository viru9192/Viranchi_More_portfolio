## üöÄ Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/viru9192/Viranchi_More_portfolio.git
   cd Viranchi_More_portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
